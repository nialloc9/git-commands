# Git Commands

This repository contains opinionated Git workflow helpers grouped by workflow style.

## Folders

- [gitflow](gitflow/README.md): Scripts and make targets for a standard Gitflow process.
- [github-flow](github-flow/README.md): Scripts and docs for a lightweight GitHub Flow process.

## Getting Started

1. Choose a workflow folder.
2. Read that folder's README.
3. Run commands from inside that folder.
